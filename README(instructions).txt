Instructions to Run the Program
Database:
1. use the mySql text file to input commands to create the database

WEB API
1. Open the ephoneshop in Visual Studios 2022 through the solution
2. Build the solution
3. Change the connectionstring in appsettings.json
3. Run

ANGULAR (frontend)
1. open the ephoneshop folder in vs code
2. make sure the path ends with 'ephoneshop/' in the terminal
3. type npm install in the terminal at the location above ^
4. type npm start (theoretically it should work)
